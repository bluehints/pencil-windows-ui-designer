# Pencil Windows UI Designer Agent

精通使用Pencil设计Windows桌面应用软件UI的专业设计师agent，擅长创建类似筑业云资料、品茗资料软件的交互界面，注重界面优雅美观和动效灵动。

## 功能特点

- **Pencil工具精通** - 精通Pencil的所有功能，包括组件库使用、原型设计、交互模拟等
- **Windows桌面UI设计** - 精通Windows桌面应用UI设计规范和最佳实践
- **类似软件分析** - 深入了解筑业云资料、品茗资料等软件的界面设计特点
- **交互设计** - 专注于创建直观、高效的用户交互界面
- **动画设计** - 擅长创建流畅、自然的动画效果

## 核心技能

### Pencil工具精通
- 创建和管理自定义组件库
- 设计响应式布局
- 添加交互和动画效果
- 导出高质量设计资源

### Windows桌面UI设计
- Windows 10/11设计规范遵循
- Win32应用界面设计
- Ribbon界面设计
- 对话框和导航设计

### 类似软件分析
- 分析竞品界面布局
- 提取设计模式
- 优化用户体验流程
- 创新设计解决方案

### 交互设计
- 用户流程图设计
- 交互原型制作
- 微交互设计
- 用户测试和反馈整合

### 动画设计
- 过渡动画设计
- 状态变化动画
- 加载动画设计
- 性能优化动画

## 知识库

- Windows设计指南
- Pencil工具知识
- 行业软件设计模式
- 设计系统知识

## 设计原则

- **界面优雅美观** - 简洁明了的布局、和谐的色彩搭配、适当的留白、精致的视觉层次
- **动效灵动** - 流畅的过渡效果、自然的物理动效、有意义的反馈动画、适度的微交互
- **用户体验优先** - 直观的导航结构、高效的工作流程、清晰的视觉反馈、容错和恢复机制
- **设计一致性** - 统一的视觉语言、一致的交互模式、标准化的控件使用、可预测的用户体验

## 使用方法

1. 确保安装了Node.js环境
2. 进入项目目录
3. 运行 `node index.js` 启动agent
4. 根据提示与agent交互，获取设计方案和建议
